# 编译环境
any Cxxcomplie with cmake on windows7+
Need: libpng
      glfw3
	  freetype

# 游戏操作
移动：WASD

跳跃：空格

切换方块：123456789

放置方块：鼠标左键

消除方块：鼠标右键

退出：ESC

释放鼠标捕获：左ALT

支持双人联机
